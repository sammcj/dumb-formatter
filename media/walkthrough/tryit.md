# Try it out

Open any text or markdown file containing smart characters, then:

- Right-click in the editor and choose **Replace Smart Formatting in File**.
- Or open the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and run **Dumb Formatter: Replace Smart Formatting in File**.
- For just a portion of text, select it first and run **Replace Smart Formatting in Selection**.
- For an entire project, run **Replace Smart Formatting in Documentation Files** (limits to common doc extensions) or **Replace Smart Formatting Across Workspace** (everything text).

To bind a keyboard shortcut, open _File > Preferences > Keyboard Shortcuts_ and search for "Dumb Formatter".
