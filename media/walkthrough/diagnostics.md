# Inline diagnostics

When enabled, Dumb Formatter highlights smart characters as you type and offers one-click quick fixes.

- Severity defaults to _information_; configurable via `dumb-formatter.diagnostics.severity`.
- Source code files are skipped unless `dumb-formatter.sourceCode.scope` is set to `comments-only` or `all`.

Toggle later via _Settings > Dumb Formatter > Diagnostics: Enabled_.
